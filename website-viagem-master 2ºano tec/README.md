# ✈️ Site da Agência de Viagens: Sua Jornada Começa Aqui!

Bem-vindo ao repositório do **Site da Agência de Viagens**, um projeto desenvolvido para apresentar uma plataforma estática e responsiva, ideal para agências que desejam uma presença online elegante e funcional. Este site foi construído com foco na experiência do usuário, utilizando as tecnologias web mais fundamentais para garantir acessibilidade e adaptabilidade em qualquer dispositivo.

## ✨ Visão Geral do Projeto

Este projeto demonstra a criação de um website de agência de viagens estático, com um design moderno e intuitivo. O principal objetivo foi explorar e aplicar conceitos essenciais de desenvolvimento web front-end, resultando em uma interface que simula a navegação por destinos, pacotes e informações de contato de uma agência de viagens.

## 🚀 Funcionalidades e Destaques

O site incorpora diversas funcionalidades e técnicas que o tornam robusto e fácil de usar:

-   **Layout Flexível com Flexbox:**
    -   A estrutura do site é construída com base nos princípios do Flexbox, utilizando propriedades como `flex-direction`, `justify-content` e `align-items` para criar layouts dinâmicos e alinhamentos precisos. Isso garante que os elementos se ajustem harmoniosamente em diferentes tamanhos de tela.

-   **Integração com Google Maps:**
    -   A seção de contato apresenta um mapa interativo, integrado diretamente com a API do Google Maps. Isso permite que os visitantes visualizem facilmente a localização da agência, melhorando a usabilidade e a acessibilidade das informações de contato.

-   **Design Responsivo (Media Queries):**
    -   O site é totalmente responsivo, adaptando-se perfeitamente a smartphones, tablets e desktops. Isso foi alcançado através do uso estratégico de Media Queries em CSS, garantindo que a experiência de navegação seja consistente e agradável, independentemente do dispositivo utilizado.

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

-   **HTML5:** Para a estruturação semântica do conteúdo.
-   **CSS3:** Para a estilização e o design responsivo.
-   **JavaScript:** Para interações dinâmicas e aprimoramento da experiência do usuário (uso mínimo para funcionalidades específicas).
